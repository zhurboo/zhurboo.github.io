LeetCode 257. Binary Tree Paths

发表于 2021-05-24 分类于 Leetcode

题目描述(英文)

Given the root of a binary tree, return all root-to-leaf paths in any order.

A leaf is a node with no children.

题目描述(中文)

给定一个二叉树，返回所有从根节点到叶子节点的路径。

说明: 叶子节点是指没有子节点的节点。

样例

题解

dfs搜索，记录当前路径并回溯。

时间复杂度 $O(n^2)$。
