apt conda pip命令

发表于 2021-03-31 分类于 配置与命令

Apt-get

  • 更新
命令 功能
apt-get update 更新源
apt-get upgrade 更新已安装的包
apt-get dist-upgrade 升级系统
  • 搜索包
命令 功能
apt-cache search 搜索包
apt-cache show 获取包的相关信息
  • 管理包
命令 功能
apt-get install 安装包
sudo apt-get install = 安装特定版本的包
apt-get -f install 修复安装包
apt-get remove 删除包
apt-get remove - - purge 删除与包相关的所有内容
apt-get clean 清理无用的包
apt-get autoclean 清理无用的包
apt-get autoremove 删除自动安装的lib和包

Conda

  • 添加源
命令 功能
conda config —add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/ 添加清华源
conda config —set show_channel_urls yes 添加源生效
  • 版本管理
命令 功能
conda —version 查看当前版本
conda update conda 升级当前版本
  • 管理虚拟环境
命令 功能
conda env list 查看所有虚拟环境
conda create -n name python=3.6 创建虚拟环境
conda create -n name1 —clone name2 克隆服你环境
conda remove -n name —all 删除虚拟环境
conda activate name 激活虚拟环境
conda deactivate 关闭虚拟环境
  • 管理包
命令 功能
conda list 查看已安装的文件包
conda install xxx=x.x 安装文件包
conda update xxx 更新文件包
conda uninstall xxx 删除文件包
conda list -e > requirements.txt 批量导出包
conda install —yes —file requirements.txt 批量安装包
conda clean -p 清除无效缓存

Pip

  • 添加源
  • 管理包
命令 功能
pip list 查看已安装的文件包
pip install xxx==x.x 安装文件包
pip install -U xxx 更新文件包
pip uninstall xxx 删除文件包
pip freeze > requirements.txt 批量导出包
pip install -r requirements.txt 批量安装包
